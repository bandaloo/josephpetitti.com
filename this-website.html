About

This website is written almost entirely in HTML and CSS. The point of this website was to help teach myself web design, and make a personal website that looks good enough to show off to employers and friends. The design of the site has evolved over several weeks as I learn of new ways to do things and think of more things that would look cool. I also sometimes use the site to host pieces of projects for classes. To see the history of changes I've made to the site, check out my GitHub page.

Unfortunately, one of the issues I've run into is that I don't know what to actually write! It turns out that creating good-looking pages is a lot easier than coming up with interesting content to put on it.

Also, I have discovered that CSS is annoying. I eventually figured out how to do everything I want to, but it took many frustrating hours. For example, these parallax headers look really cool, but it took a very look time to figure out how to position everything properly.
